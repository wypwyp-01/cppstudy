# C++ 学习代码集合

本仓库是我在学习 C++ 编程过程中所编写和整理的一些示例代码，涵盖了基础语法、算法练习、IO 多路复用、网络编程等多个方面，方便日后复习和参考。

##  目录说明

- `sort/`  
  常见排序算法实现，如冒泡、快速排序、归并排序等。

- `shi01/`  
  施磊老师的 C++ 初级课程相关代码，涵盖基础语法与入门案例。

- `shi02/`  
  施磊老师的 C++ 高级课程内容，包括 STL、类与对象进阶等。

- `iomulti/`  
  IO 多路复用相关内容，包含 `select`、`poll`、`epoll` 的示例代码。

- `computernet/`  
  使用 socket 进行网络编程的相关练习，如 TCP/UDP 通信示例等。

##  使用方式

本项目无需额外依赖，直接使用支持 C++11 或更高标准的编译器（如 g++）即可进行编译和测试：

```bash
g++ example.cpp -o example
./example
